Exercise 2: Video

Video is one of the defining features of web development. Let's get some practice with some classics.

Home

Inside the section with an id of answers, create a video element. Please use the video file Charlie_Bit_My_Finger.mp4. Use the controls, loop, and autoplay attributes. Videos should be given a height and width as well. Some browsers will not autoplay without the video being muted.

Inside the section with an id of answers, create a video element. Please use the video file Dramatic_Chipmunk.mp4. Use the controls, muted, width, height, and poster attributes. You can choose with dimensions of the video. The poster attribute is the thumbnail image it will display before the video is played. The poster attribute should be in the video tag, not the source tag.

Inside the section with an id of answers, create an iframe. Please link the following video in its src attribute: https://www.youtube.com/embed/QH2-TGUlwu4.
